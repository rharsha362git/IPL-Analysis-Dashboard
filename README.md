# 🏏 IPL Analysis Dashboard (2008–2025)

## 📌 Project Overview
This project is an interactive Power BI dashboard that analyzes IPL data from 2008 to 2025. It provides season-wise insights into team performance, player statistics, and tournament highlights.

---

## 🚀 Features
- Season-wise analysis
- Team standings
- Orange Cap & Purple Cap statistics
- Total Matches, Fours, Sixes, Centuries & Half-Centuries
- Interactive filters and slicers

---

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📊 Dataset
- IPL Matches
- Player Statistics
- Team Information
